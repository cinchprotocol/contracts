# Revenue Share Vault

## Development
```
yarn install
yarn compile
yarn test
```

## Deployment
* Set the corresponding env variables as in .envrc.example
* Then run yarn deploy with specific tags of deploy script, i.e.
```
yarn deploy --tags RevenueShareVaultRibbonEarnMock
```
